# gin-errors
The best implementation of error handling in gin

[中文文档](./README-zh.md)

## Installation


``` shell
git clone https://github.com/Darr-en1/gin-errors.git
```



### Download dependencies

```shel
cd gin-errors && go mod tidy
```



### Run

```golang
go run demo/main/main.go
```
